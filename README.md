# 🎓 AI-Based Student Career Path Recommender

This project is a **Data Science based web application** that recommends suitable career paths to students based on their skills and interests.

---

## 🚀 Project Description

The system analyzes user input (skills/interests) and matches it with a dataset of career options using machine learning techniques. It then suggests the most relevant career paths along with a learning roadmap.

---

## 🧠 Features

- Career recommendation based on skills
- Top 3 career suggestions
- Learning roadmap for each career
- Simple and interactive web interface
- Beginner-friendly AI/ML project

---

## 🛠️ Technologies Used

- Python  
- Streamlit  
- Pandas  
- Scikit-learn (TF-IDF + Cosine Similarity)

---

## 📊 How It Works

1. User enters skills or interests  
2. System converts text into numerical features (TF-IDF)  
3. Similarity is calculated using cosine similarity  
4. Top matching careers are recommended  
5. Roadmap is displayed for each career
